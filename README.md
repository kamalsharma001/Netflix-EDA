# Netflix Movies & TV Shows – Exploratory Data Analysis

I recently worked on an EDA project using the Netflix Movies &amp; TV Shows dataset to explore patterns around content type, ratings, duration, and country-wise distribution.

This project performs exploratory data analysis on the Netflix Movies & TV Shows dataset to understand content distribution, ratings, duration trends, and country-wise contributions.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Movies dominate the Netflix catalog compared to TV shows.
- Most movies are 80–120 minutes long.
- TV shows are mostly short-running (1–2 seasons).
- Mature ratings such as TV-MA and TV-14 are most common.
- The US is the largest content producer on Netflix.

## Dataset
Netflix Movies & TV Shows dataset (Kaggle)
